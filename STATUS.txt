1. TWO-VEHICLE INTERSECTION ENVIRONMENT
   📁 env/intersection_env.py
   - Gymnasium-compatible interface
   - Two vehicles approaching from perpendicular directions
   - Continuous observation and action spaces
   - Configurable parameters (time step, max steps, goal threshold)

2. DETERMINISTIC VEHICLE DYNAMICS
   📁 env/dynamics.py
   - Kinematic model with position and velocity
   - Realistic acceleration/braking constraints
   - Collision detection using bounding boxes
   - Max speed: 15 m/s, Max accel: 3 m/s², Max brake: 5 m/s²

3. REWARD STRUCTURE
   📁 env/rewards.py
   - Collision penalty: -100
   - Goal reward: +50
   - Time penalty: -0.1/step
   - Efficiency bonus for maintaining speed
   - Proximity penalty for getting too close

4. HEURISTIC BASELINE AGENTS
   📁 agents/heuristic.py
   - ConstantVelocityAgent: Maintains steady speed
   - CautiousAgent: Slows down when other vehicle nearby
   - AggressiveAgent: High speed, emergency brakes only
   - YieldAgent: Yields to other vehicle at intersection

5. VISUALIZATION & DEMO
   📁 demo/render_episode.py
   - Run episodes with any agent pair
   - Plot vehicle trajectories (2D)
   - Display speed over time
   - Show episode statistics
   - Compare multiple scenarios

💡 EXAMPLE USAGE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

from env import IntersectionEnv
from agents import create_heuristic_pair
import numpy as np

# Create environment and agents
env = IntersectionEnv()
agent_1, agent_2 = create_heuristic_pair("cautious", "aggressive")

# Run episode
obs, info = env.reset()
done = False

while not done:
    action = np.array([
        agent_1.get_action(obs["agent_1"]),
        agent_2.get_action(obs["agent_2"])
    ])
    obs, reward, terminated, truncated, info = env.step(action)
    done = terminated or truncated

print(f"Collision: {info['collision']}")
print(f"V1 reached goal: {info['done_1']}")
print(f"V2 reached goal: {info['done_2']}")
